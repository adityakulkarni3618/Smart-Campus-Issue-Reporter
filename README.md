# Smart Campus Issue Reporter

## Problem Statement
Campus-related issues such as infrastructure damage, cleanliness, and safety concerns lack a centralized digital reporting system. This results in delayed resolutions and poor accountability.

## Proposed Solution
Smart Campus Issue Reporter is a web-based platform that allows students to report campus issues digitally and track their resolution status in real time. Campus authorities can manage and update issues through an admin dashboard, improving transparency and response time.

## Key Features
- User authentication
- Issue reporting with category and description
- Real-time status tracking
- Admin dashboard for issue management

## Technologies Used
